# Úlohy ze cvičení z předmětu KIV/PPA

## O repozitáři
Tento repozitář obsahuje vypracované úlohy ze cvičení z předmětu KIV/PPA v akademickém roce 2025/2026. Úlohy se můžou lišit (a liší) v závislosti na cvičícím.

## Zdroje
Všechny zdroje jsou dostupné po přihlášení se k účtu Orion [zde](https://programovani.kiv.zcu.cz/).
